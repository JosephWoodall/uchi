# Security Policy

## Supported Versions

We currently support the following versions of Uchi with security updates.

| Version | Supported          |
| ------- | ------------------ |
| >= 0.2.x| :white_check_mark: |
| < 0.2.0 | :x:                |

## Reporting a Vulnerability

We take the security of Uchi very seriously. If you discover a security vulnerability, please do NOT report it by opening a public GitHub issue. 

Instead, please send an email privately to the repository maintainer. We will acknowledge receipt of your vulnerability report and strive to send you regular updates about our progress in resolving it. If the vulnerability is accepted, we will coordinate a public disclosure along with the release of the patch.
